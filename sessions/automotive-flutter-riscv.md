# Accelerating Open Automotive Innovation: Flutter on RISC-V

https://www.socallinuxexpo.org/scale/23x/presentations/accelerating-open-automotive-innovation-flutter-risc-v

- **Date**: Saturday, March 7, 2026
- **Time**: 17:00 - 18:00
- **Location**: Room 211
- **Speaker(s)**: Yuning Liang, Founder & CEO, DeepComputing

> "As AI automotive moves rapidly toward open and modular software architectures, RISC-V offers a unique path for transparent, customizable compute platforms too. Flutter is becoming one of the most cross-platform new and respectable application frameworks as good as Android, with fewer resources required and less technical debt. This session highlights our journey enabling all 3 new trends altogether, Flutter for AI Automotive experiences on RISC-V hardware. We explore the open-source components, performance optimizations, and system design needed to support responsive HMIs, and discuss how the community can contribute to building a fully open RISC-V AI automotive software ecosystem with Flutter popularity."

## Overview

A journey into enabling Flutter-based automotive HMI experiences on RISC-V hardware, exploring the open-source components and performance optimizations needed for a fully transparent, community-driven AI automotive software stack.

## Key Points

- x86 vs ARM vs RISC-V
- RISC-V most Linux-friendly ISA
- Flutter originally from Google
- Android, Linux, iOS, cross-platform
- AGL release in 2025
- Flutter Embedded for Automotive, AGL
- Flutter AI framework for local compute in progress
