# Sponsored Workshop: From Tables to Streams: Apache Flink for SQL Developers

https://www.socallinuxexpo.org/scale/23x/presentations/sponsored-workshop-tables-streams-apache-flink-sql-developers

- **Date**: Friday, March 6, 2026
- **Time**: 10:00 - 13:00
- **Location**: Room 207
- **Speaker(s)**: Viktor Gamov, Principal Developer Advocate, Confluent; Adi Polak, Director of Advocacy, Confluent

> What if your SQL queries could process data as it happens instead of after it's stored? Apache Flink speaks SQL fluently, but it's not a database – think of it as a conductor orchestrating endless streams of data rather than a librarian managing stored records.
>
> The session covers Flink SQL operations on infinite data streams, the Table API for programmatic control, real-time feature engineering via Flink AI, and practical applications including fraud detection during transactions, live analytics dashboards, and event-driven architectures.

## Overview

A hands-on workshop introducing SQL developers to Apache Flink for real-time stream processing, covering the Table API, real-time feature engineering, and practical streaming use cases like fraud detection and live analytics.
