# OpenClaw: The Non-Engineer's Survival Guide

https://www.socallinuxexpo.org/scale/23x/presentations/openclaw-non-engineers-survival-guide

- **Date**: Saturday, March 7, 2026
- **Time**: 12:30 - 13:30
- **Location**: Ballroom DE
- **Speaker(s)**: Matt Ramage, Founder, Emarketed

> "I'm not a developer and I'm not a hardcore Linux user, but I still got OpenClaw running and useful fast. In this talk, I'll show the real workflows I use, the mistakes I made (including a $400 token loop), and the shortcuts that actually work. If you want a practical guide to getting a useful agent setup without the pain, this is for you."
>
> The speaker covers setup options (cloud vs. local), security considerations, skills development, common errors, and Telegram/Discord integration. Attendees will gain practical understanding of OpenClaw's capabilities and learn to deploy it quickly.

## Overview

A practical, experience-first guide for non-engineers to set up and effectively use the OpenClaw AI agent framework, sharing real workflows, hard-won lessons, and integrations that make it immediately useful.
